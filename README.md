# 👕 Sistema de Controle de Estoque - Loja de Roupas

Este é um sistema desenvolvido para simplificar e automatizar o gerenciamento de inventário e o fluxo de saída de mercadorias de uma loja de roupas.

## 🚀 Funcionalidades Principais

* **Gestão de Inventário:** Cadastro de peças com especificações de tamanho, cor e categoria.
* **Controle de Quantidade:** Atualização em tempo real do volume de itens disponíveis.
* **Monitoramento de Envios:** Registro e acompanhamento das mercadorias enviadas aos clientes.
* **Alertas de Reposição:** Avisos automáticos quando uma peça atinge o estoque mínimo.

## 🛠️ Tecnologias Utilizadas

- **Spring Web**: Criação da aplicação web estruturada no padrão MVC.

- **Thymeleaf**: Motor de templates para renderização do HTML dinâmico no servidor.

- **Spring Data JPA**: Abstração e facilitação das operações de banco de dados com Hibernate.

- **MySQL Driver**: Conector oficial para comunicação com o banco de dados MySQL.

- **Spring Boot DevTools**: Reinicialização automática da aplicação para agilizar o desenvolvimento.

- **Lombok**: Redução de código repetitivo (boilerplates) através de anotações como `@Data`.
